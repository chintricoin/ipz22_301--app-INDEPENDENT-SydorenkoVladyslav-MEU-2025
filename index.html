Самостійна Робота 1

1. Що таке HTML?

HTML (HyperText Markup Language) – це основна мова для створення веб-сторінок. Вона використовується для структурування контенту і надання йому базової форми.

Приклад базової структури HTML-документа:

<!DOCTYPE html>
<html>
  <head>
    <title>Назва сторінки</title>
  </head>
  <body>
    <h1>Тарас Бульба</h1>
    <p>Це приклад HTML сторінки.</p>
  </body>
</html>

2. HTML Редактори коду та IDE

HTML можна писати у будь-якому текстовому редакторі, але для зручності використовуються спеціальні редактори коду (IDE), наприклад Visual Studio Code, Atom, або Sublime Text. Вони спрощують написання коду завдяки підсвічуванню синтаксису та автодоповненню.

3. HTML Основні приклади

Це простий приклад HTML сторінки з використанням заголовків, параграфів та списків.

<!DOCTYPE html>
<html>
  <body>
    <h1>Заголовок сторінки</h1>
    <p>Це приклад абзацу тексту.</p>
    <ul>
      <li>Елемент списку 1</li>
      <li>Елемент списку 2</li>
    </ul>
  </body>
</html>

4. HTML Елементи

HTML елементи складаються з відкриваючого та закриваючого тегів, між якими знаходиться вміст. Наприклад:

<p>Це абзац.</p>
<a href="https://example.com">Це посилання</a>

5. HTML Атрибути

Атрибути надають елементам додаткову інформацію. Наприклад, атрибут href вказує на адресу для посилання, а alt – на альтернативний текст для зображень.

<a href="https://example.com">Перейти на сайт</a>
<img src="image.jpg" alt="Опис зображення">

7. HTML Посилання

Посилання (елемент <a>) використовуються для переходу на інші сторінки. Атрибут href задає URL цілі.

<a href="https://example.com">Перейти</a>
        Посилання на приклад

8. HTML Зображення

Зображення вставляються за допомогою тегу <img> з атрибутом src (посилання на файл зображення) та alt (опис зображення).

<img src="C:\Users\LENOVO\Pictures\Screenshots\cat.png" alt="Приклад зображення">

9. HTML Таблиці

Таблиці використовуються для представлення даних у структурованому вигляді, з рядками та стовпцями.

Назва Ціна
Продукт 1 100 грн
Продукт 2 200 грн

10. HTML Списки

HTML має різні типи списків, наприклад:

Невпорядкований список:

  • Елемент 1
  • Елемент 2
  • Елемент 3

Впорядкований список:

  1. Елемент 1
  2. Елемент 2
  3. Елемент 3
